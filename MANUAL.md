# LSPU Medicine Inventory System - User Manual

## Welcome to the System! 
This guide will help you understand how to use the LSPU Medicine Inventory System effectively. As a normal user, you have access to various features that will help you manage and track medicine transactions.

---

## Getting Started 

### Logging In
1. Open your web browser and go to the system's login page.
2. Enter your username and password.
3. Click the **"Login"** button.
4. If successful, you'll be directed to the dashboard.

### Viewing Medicine Inventory
1. Click on **"Inventory"** in the navigation menu.
2. You'll see a list of all available medicines with details such as:
   - Medicine name
   - Current stock
   - Expiration dates
   - Categories
3. Use the search bar to quickly find specific medicines.
4. Sort columns by clicking on the column headers.
5. **Note**: As a normal user, you have view-only access to the medicine inventory. Adding or editing medicines requires admin privileges. If you notice any discrepancies in medicine information, please contact your administrator.

---

### Managing Medicine Inventory
1. Click on **"Inventory"** in the navigation menu.
2. You'll see a list of all available medicines with details such as:
   - Medicine name
   - Current stock
   - Expiration dates
   - Categories

#### Adding New Medicine
1. Click the **"Add New Medicine"** button.
2. Fill in the required information:
   - Medicine name
   - Generic name
   - Brand name (if applicable)
   - Category
   - Initial stock quantity
   - Unit (e.g., tablets, bottles)
   - Expiration date
   - Description (optional)
3. Click **"Save"** to add the medicine to inventory.

#### Editing Medicine Information
1. Find the medicine you want to edit in the inventory list.
2. Click the **"Edit"** button (pencil icon) next to the medicine.
3. Update any of the following:
   - Medicine details
   - Stock quantity
   - Expiration date
   - Category
   - Description
4. Click **"Save Changes"** to update the medicine information.

#### Deleting Medicine
1. Locate the medicine in the inventory list.
2. Click the **"Delete"** button (trash icon).
3. Confirm the deletion when prompted.
4. **Note**: Only delete medicines that were added by mistake. For out-of-stock items, update the quantity instead.

#### Managing Inventory
- Use the search bar to quickly find specific medicines
- Sort columns by clicking on the column headers
- Filter medicines by category or status
- Export inventory reports when needed
- Track low stock items
- Monitor expiring medicines

#### Best Practices for Inventory Management
- Regularly update stock quantities
- Double-check information when adding new medicines
- Keep descriptions clear and consistent
- Document any special storage requirements
- Update expiration dates when receiving new stock
- Report any discrepancies immediately

---

### Managing Transactions
1. Navigate to **"Transactions"** in the navigation menu.
2. Click **"New Transaction"** to create a new entry.
3. Fill in the required information:
   - Date and time
   - Patient name
   - Course/Year/Section
   - Patient complaints
   - Selected medicine(s)
   - Quantity dispensed
   - Any additional remarks
4. Review the details.
5. Click **"Submit"** to complete the transaction.

#### Transaction Status Management
- For ongoing transactions:
  - Click **"Finish"** to mark a transaction as complete
  - Click **"Cancel"** if the transaction needs to be voided
  - Add time finished automatically when completing a transaction

#### Transaction Categories
You can view transactions in two categories:
- **Ongoing Transactions**: Currently active patient visits
- **Finished Transactions**: Completed patient consultations

---

### Viewing Transaction History
1. Navigate to **"Transactions" > "List"**.
2. On this page, you can:
   - View all your past transactions
   - Filter transactions by status (ongoing/finished)
   - Search for specific transactions
   - Sort by date, patient name, or status
   - View complete transaction details including medications dispensed

---

### Profile Management
1. Click on your profile icon in the top-right corner.
2. Select **"Profile Settings"** to:
   - Update your personal information
   - Change your password
   - Manage notification preferences
   - View your user role and permissions
